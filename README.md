# Luxury Real Estate Website

A beautiful, modern luxury real estate website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Modern, luxury design aesthetic
- Responsive layout
- Beautiful typography with Playfair Display and Inter fonts
- High-quality property images
- Smooth animations and transitions

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project can be deployed to any platform that supports Next.js, such as:
- Vercel (recommended)
- Netlify
- AWS Amplify
- Any Node.js hosting service

For subdomain deployment, configure your DNS settings to point your subdomain to your hosting provider.
